Electron framework comes to help for this need. There are some alternatives available but the one we will test is electron-builder.
First of all we need to install electron-builder to our project. Because weâ€™re using npm as the package manager for our project, in a terminal weâ€™ll issue the command

  "build": {
    "appId": "com.maurizioattanasi.electron-hmi",

      "target": "nsis",

and a dist command in the script section to build and distribute the application itself
...

    "electron": "ng build && electron .",
    "electron-aot": "ng build --aot && electron .",
    "dist": "build"

...

As it can be seen in the build section we have three subsections win, mac and linux meaning that, depending on the platform on which weâ€™re developing our app we can have an appropriate installer.
